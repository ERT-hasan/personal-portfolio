# 💼 Personal Portfolio - React + Vite + TailwindCSS

This is a personal portfolio web application built using **React**, bundled with **Vite**, and styled using **Tailwind CSS**.  
The project is configured with **ESLint** for code quality and supports modern tooling like **Framer Motion**, **Heroicons**, and **React Icons**.

---

## ✅ What Has Been Done in This Project

- 🔹 **React 19 Setup** with Vite for fast and modern development experience.
- 🔹 **Tailwind CSS 4** integrated and customized with a dark mode (`class` strategy) and custom color palette.
- 🔹 Created a structured layout with a clean folder structure and maintainable code.
- 🔹 Integrated **Heroicons**, **Framer Motion**, and **React Icons** for professional UI/UX.
- 🔹 Configured **ESLint** with React rules to ensure clean and error-free code.
- 🔹 Added Dockerfile for optional Docker support.
- 🔹 Used `vite.config.js` to automatically open the project in the browser on dev start.
- 🔹 Ensured responsive design with mobile-first approach.
- 🔹 SEO basics handled via `index.html` (`lang`, `viewport`, `title`).


---

## 📁 Key Files

- `index.html` – Entry point of the app
- `src/` – React components and logic
- `tailwind.config.js` – Tailwind custom config
- `vite.config.js` – Vite dev server setup
- `eslint.config.js` – Linting rules
- `Dockerfile` – For Docker-based deployment
- `package.json` – Project scripts and dependencies
